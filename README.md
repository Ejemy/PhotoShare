# Photo Share
A website for uploading and previewing photos.
写真のアップロードとプレビューのためのウェブサイトです。


## Features
- Upload, delete, and preview photos from any computer.

## 特徴
-どのコンピュータからでも写真をアップロード、削除、およびプレビューできます。

## Dependencies
- googleapis
- express
- nodejs
